# ClaimKompas site v3 (static)

Deze versie is **extra toegankelijk** en **heel duidelijk** voor verzekeringsmaatschappijen:
- minimale tekst
- heldere 3-staps workflow (triage → precedent → besluit)
- product in 3 tabs (triage / dossier / rapport)
- animaties subtiel + respecteert `prefers-reduced-motion`
- extra focus op toegankelijkheid (skip link, focus-states, keyboard tabs)

## Starten
1. Pak de zip uit.
2. Open `index.html` in je browser.
